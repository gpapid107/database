###Καλωσορίσατε στο Νοσοκομείο "Υγειόπολις"!

Σας παρουσιάζουμε μια Βάση Δεδομένων με ρεαλιστικά δεδομένα ασθενών, τμημάτων, προσωπικού, νοσηλειών, και ό,τι αφορά ένα νοσοκομείο. Μάλιστα, για δεδομένα όπως τα φάρμακα, οι ιατρικές πράξεις, οι κωδικοί ΚΕΝ και ICD-10 χρησιμοποιούμε πραγματικά δεδομένα, αναγνωρισμένα από το Ελληνικό Κράτος. Αφού ένας ασθενής φτάσει στο νοσοκομείο, παραπέμπεται σε διαλογή και μπαίνει σε σειρά προτεραιότητας, ανάλογα με τη σοβαρότητα του περιστατικού προς αντιμετώπιση και δευτερευόντως με το χρόνο άφιξης. Μόλις έρθει η σειρά του, λαμβάνεται η απόφαση για παραπομπή του σε νοσηλεία, ή αποχώρηση, αν η κατάσταση το επιτρέπει, στην οποία περίπτωση δίνονται κατάλληλες οδηγίες.

Όσον αφορά το προσωπικό, υπάρχουν αυστηροί κανόνες ως προς τις καλυπτόμενες βάρδιες που επιτρέπονται, τα δικαιώματα/καθήκοντα του καθενός και τις σχέσεις εποπτείας. Σαφώς, αυτές οι σχέσεις δεν επιτρέπεται να είναι κυκλικές. Κάθε τμήμα διαθέτει υποχρεωτικά διευθυντή, ο οποίος είναι ιατρός στο ίδιο τμήμα.

Στο repository αυτό θα βρείτε τα ER και Relational διαγράμματα, την υλοποίηση της Βάσης Δεδομένων σε περιβάλλον MySQL μαζί με το schema και τα triggers που διασφαλίζουν την ακεραιότητα και τη λειτουργική ευρωστία της. Επίσης, υπάρχει ένα σχετικά μεγάλο αρχείο δεδομένων, σε ικανοποιητικό βαθμό συνεπή με όλους τους πραγματικούς και υποθετικούς περιορισμούς ενός νοσοκομείου. Καλύπτουν ρεαλιστικά τη λειτουργία του νοσοκομείου για ένα διάστημα περίπου 2.5 χρόνων, ως προς τις νοσηλείες και τις βάρδιες.

Για να φανεί πρακτικά χρήσιμη η Βάση Δεδομένων, εκτελούμε ένα σύνολο ερωτημάτων (queries) πάνω σε αυτή και ελέγχουμε την ορθότητα των απαντήσεων. Οι αντίστοιχες απαντήσεις παρατίθενται σε αρχείο QXX.txt με QXX.sql το αντίστοιχο query.

Τέλος, ο SQL σκελετός ενσωματώθηκε σε μια διαδικτυακή εφαρμογή (Web UI) προς κομψότερη απεικόνιση και αλληλεπίδραση με τη Βάση. Παρακάτω παρατίθενται οι απαραίτητες οδηγίες για εγκατάσταση και χρήση της εφαρμογής.

## Web UI

Η web εφαρμογή αναπτύχθηκε σε Python (Flask) και παρέχει γραφικό περιβάλλον
διαχείρισης της βάσης δεδομένων του Νοσοκομείου Υγειόπολης.

### Προαπαιτούμενα
- Python 3
- MySQL Server
- Η βάση mydb φορτωμένη (install.sql → load.sql → triggers.sql)

### Εγκατάσταση & Εκτέλεση

1. Ανοίξτε terminal και πηγαίνετε στον φάκελο ui:
   cd ui

2. Φτιάξτε εικονικό περιβάλλον και εγκαταστήστε τις βιβλιοθήκες:
   
   python3 -m venv venv

   source venv/bin/activate
   
   pip install -r requirements.txt

3. Αν η MySQL σας έχει κωδικό, τρέξτε: export DB_PASSWORD='ο_κωδικός_σας'

4. Εκτελέστε:
   python app.py

5. Ανοίξτε στον browser τη διεύθυνση που εμφανίζεται στο terminal.
